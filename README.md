## 🌟 Overview

**ApexDevs** is an interactive, modern portfolio & studio application built with **React**, **Vite**, **Three.js**, **WebGL**, **GSAP**, and **Framer Motion**. It showcases our core services, shipped production case studies, team expertise, interactive bento grids, and direct contact channels.

---

## 🚀 Key Features & Interactive Components

- **⚡ MagicBento Grid**: Interactive spotlight cursor tracking, border glow, star particle effects, and 3D tilt cards highlighting studio capabilities.
- **💼 Case Studies Stack**: 3D interactive `<CardSwap />` stack presenting live projects with Problem, Solution, Tech Stack, and Results metrics.
- **📱 Responsive Mobile Navigation**: Staggered slide-out menu panel with smooth GSAP underlayer transitions.
- **👥 Expandable Team Showcase**: Interactive team cards with click-to-expand skill breakdowns and direct Telegram, GitHub & LinkedIn links.
- **⭐ Testimonials Marquee**: Animated 5-star endorsement cards with smooth infinite loop scrolling.
- **🔊 Interactive Audio System**: Ambient section soundscapes and responsive UI click & hover sound effects with header toggle controls.
- **🎨 Glassmorphism & WebGL Aesthetics**: Shader-based 3D background elements, custom glowing borders, and target cursor follower.

## 🛠️ Local Development & Getting Started

### 1. Prerequisites
- Node.js (v18.0.0 or higher)
- npm or yarn

### 2. Installation
```bash
# Clone repository
git clone https://github.com/shahzodislomov/apex.git

# Navigate into project directory
cd apex

# Install dependencies
npm install
```

### 3. Start Development Server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.

### 4. Production Build
```bash
npm run build
```
